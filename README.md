# PhoneBook Management System

## Overview
This is a simple console-based PhoneBook Management System written in C++.  
The program allows users to store and manage contacts using a menu-driven interface.

Users can:
- Add contacts
- Display all contacts
- Remove contacts
- Edit existing contacts
- Search for contacts by name

The application uses:
- Classes and objects
- Vectors
- Functions
- String handling
- Basic searching and editing functionality
